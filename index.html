🎯 Що ми вивчимо сьогодні?

Сьогоднішній урок — це ваш квиток у світ веб-розробки. Ми розберемося з трьома головними технологіями, які працюють у кожному сайті.

HTML — це не програмування! 🤯

Багато хто думає, що створення сайтів — це складне програмування. Насправді HTML — це розмітка, як у Word-документі. Ви просто позначаєте: "це заголовок", "це параграф", "це зображення".

📄 HTML — Каркас будинку

Уяви будинок 🏠

HTML — це як каркас будинку: стіни, дах, фундамент. Без цього немає чого прикрашати. HTML створює структуру сторінки.

Базова структура HTML-документа:

<!DOCTYPE html> <html> <head> <title>Моя сторінка</title> </head> <body> <!-- Тут весь вміст, який бачить користувач --> </body> </html>
  • <!DOCTYPE html> — каже браузеру, що це HTML5
  • <html> — корінь документа
  • <head> — службова інформація (назва, стилі)
  • <body> — те, що бачить користувач

Основні теги HTML:

<h1> - <h6>

Заголовки — від найбільшого (h1) до найменшого (h6)

<h1>Привіт, світ!</h1>

Використовується для назв статей, розділів, підзаголовків

<p>

Параграф — блок тексту

<p>Це мій перший абзац.</p>

Для основного тексту, статей, описів

<img>

Зображення — вставка картинки

<img src="photo.jpg" alt="Моє фото">

Атрибут src — шлях до файлу, alt — опис для незрячих

<a>

Посилання — клікніть і перейдете

<a href="https://google.com">Google</a>

Для навігації між сторінками, завантажень

<div>

Контейнер — групує елементи

<div class="menu">...</div>

Для створення блоків, секцій, колонок

<br>, <hr>

Перенесення рядка та лінія

Текст<br>Новий рядок<hr>Лінія

Для форматування тексту, розділення

🎨 CSS — Дизайн та стиль

CSS = Одяг та інтер'єр 👔

Якщо HTML — це скелет будинку, то CSSCascading Style Sheets — каскадні таблиці стилів — це фарби, шпалери, меблі, декор. CSS робить сайт красивим!

Що можна змінювати за допомогою CSS:

🎨 Кольори

color: red;
background: blue;

✏️ Шрифти

font-size: 20px;
font-family: Arial;

📐 Розміри та відступи

width: 300px;
margin: 20px;

✨ Ефекти

border-radius: 10px;
box-shadow: 0 4px 6px gray;
/* CSS код */ h1 { color: #6366f1; /* Колір тексту */ font-size: 32px; /* Розмір шрифту */ text-align: center; /* Вирівнювання */ background: #f0f9ff; /* Фон */ padding: 20px; /* Внутрішній відступ */ border-radius: 12px; /* Закруглення кутів */ }

⚡ JavaScript — Життя та рух

JavaScript = Електрика та автоматика ⚡

HTML і CSS створюють статичну сторінку. JavaScriptМова програмування для веб-сторінок додає поведінку: кнопки реагують на натискання, з'являються анімації, все стає інтерактивним!

Що вміє JavaScript:

  • 🖱️ Реагувати на дії — кліки миші, натискання клавіш, прокрутка
  • 📝 Змінювати контент — текст змінюється без перезавантаження сторінки
  • 🎭 Анімації — плавні переходи, рухомі елементи, ігри
  • 🧮 Обчислення — калькулятори, перевірка форм, тести

Живий приклад:

Натисни кнопку нижче! 👇

// JavaScript код function changeText() { document.getElementById('demo-text').innerHTML = 'Текст змінено! 🎉'; } function changeColor() { document.getElementById('demo-text').style.color = '#ec4899'; }
